
# Saudi Destination - وجهة السعودية

منصة سياحية سعودية لتنظيم الرحلات الداخلية وربط الزوار بالمرشدين السياحيين المعتمدين.

## الميزات
- صفحة رئيسية جذابة
- صفحة عرض الرحلات المحجوزة
- نظام تسجيل دخول وتسجيل جديد
- لوحة تحكم للمشرف والمرشد السياحي
- تقييمات تصاعدية من العملاء
- محادثة مباشرة بين الزوار والمرشدين والمشرف
- صفحة دفع آمنة
- عداد زوار الموقع (من 200 في أبريل إلى 2000 في يونيو)
- دعم كامل للغة العربية
- تصميم عصري متجاوب

## التشغيل
```bash
pip install -r requirements.txt
python main.py
```
